# Detailed Design

## Overall Style
The website will have a clean and modern design.
- Color theme: it should look like when you are using dark theme in your code editor, this best suits my background.
- Vibe: It should give a geeky and professional vibe.
- Typography: Use a clear and readable font for the main content, there should be a very clear contrast between different levels of headings, sections, and body text. 

## Sidebar
A sidebar on the left side of the page that includes (From top to bottom):
   - A profile picture of myself.
   - My name
   - Two buttons: "中文" and "English" for language selection.
   - A navigation menu with blocks linking to different chapters of the website, including "About Me", "Experiences", "Skills", "Publications", "Contact & More". Each block should have a hover effect to indicate that it is clickable, and each will have a different gradient background color to make it visually appealing.

## Main Content Area

On the top of the main content area, there will be a header with huge and thick font displaying the current chapter name. 

The main content area will display the content corresponding to the selected chapter from the sidebar. The content will be organized into sections with clear headings and subheadings for easy navigation. The sections and subsections should be collapsible.
